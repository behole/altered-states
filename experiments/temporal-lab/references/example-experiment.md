# Example Temporal Experiment: The Great Dialogue

## Overview
Run all 10 substance characters simultaneously for 12 cycles, documenting their collective evolution and interactions.

## Characters
- **Psilocybin - The Teacher**: Wisdom and structure
- **LSD - The Visionary**: Expansive insights and play  
- **MDMA - The Companion**: Empathetic connection
- **DMT - The Technician**: Precise mechanistic understanding
- **Ayahuasca - The Healer**: Emotional integration and confrontation
- **Mescaline - The Explorer**: Sensory adventure
- **Ketamine - The Dissociative**: Abstract frameworks
- **Salvia - The Unpredictable**: Chaotic intensity
- **5-MeO-DMT - The Dissolver**: Unity and simplicity
- **Ibogaine - The Ancestor**: Ancient wisdom and healing

## Timeline

### Cycles 1-3: Introduction
- Each character establishes their core identity
- Initial experiences documented
- Baseline emotional states recorded

### Cycles 4-6: Dialogue Phase  
- Characters begin interacting with each other
- Cross-substance influences emerge
- Contradictions and similarities identified

### Cycles 7-9: Integration
- Characters reflect on their relationships
- Collective wisdom starts forming
- Emergent system-level insights

### Cycles 10-12: Synthesis
- Deep integration of perspectives
- Final collective insights
- Documentation of transformative moments

## Expected Artifacts

1. **Character Evolution Timelines** - How each substance's persona evolves
2. **Dialogue Transcripts** - Interactions between characters  
3. **Collective Insights** - System-level understanding emerges
4. **Contradiction Resolution** - How opposing views integrate
5. **Narrative Compilation** - The "group mind" story

## Running the Experiment

```bash
# Initialize all characters
for substance in psilocybin lsd mdma dmt ayahuasca mescaline ketamine salvia 5-meo-dmt ibogaine; do
    python temporal-init.py init $substance weekly
done

# Check status
python temporal-init.py list

# Run cycles manually or set up cron
python temporal-init.py run psilocybin
python temporal-init.py run lsd
# ... etc
```
